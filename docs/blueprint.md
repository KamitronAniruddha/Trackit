# **App Name**: NeetProgress

## Core Features:

- User Authentication: Secure login and signup functionality to protect user data and personalize the experience.
- Syllabus Tracking - Physics: Enable users to mark their progress on each chapter in the NEET Physics syllabus.
- Syllabus Tracking - Chemistry: Enable users to mark their progress on each chapter in the NEET Chemistry syllabus.
- Syllabus Tracking - Biology: Enable users to mark their progress on each chapter in the NEET Biology syllabus.
- Question Attempt Tracking: Allow users to log the number of questions attempted from each chapter to monitor their practice and understanding.
- Progress Analysis and Recommendations: Use AI tool to analyze the user's syllabus completion and question attempt data and recommend focused study areas, leveraging an LLM to help identify weak points and areas for improvement.

## Style Guidelines:

- Primary color: Deep blue (#3F51B5) to evoke trust and intelligence.
- Background color: Very light gray (#F0F0F0) for a clean and modern interface.
- Accent color: Green (#4CAF50) for progress indicators and positive feedback.
- Body and headline font: 'Inter', a sans-serif font for clear readability and a modern look.
- Use consistent and intuitive icons for each subject and feature. Icons should be modern and simple.
- Implement a clean, card-based layout to organize information effectively and provide a user-friendly experience.
- Incorporate subtle animations for feedback and transitions to enhance the user experience.
